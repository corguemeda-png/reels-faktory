# AI Reels Factory


AI-система, которая создаёт готовые Reels/Shorts под ключ на основе одной темы.


## Как работает
1. Вводишь тему (например: «Как похудеть к лету»)
2. Модули AI генерируют:
- идею
- структуру
- сценарий
- стиль оформления
- титры/субтитры
3. Получаешь готовый сценарий Reels.


## Папки
- templates — шаблоны
- ai_modules — логика ИИ
- assets — музыка, видео
- examples — примеры
